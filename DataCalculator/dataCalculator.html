About Data Units

Decimal Units

Decimal prefixes use powers of 1000: 1 KB = 1000 bytes, 1 MB = 1000 KB, etc. These are commonly used in data transmission and storage marketing.

Binary Units

Binary prefixes use powers of 1024: 1 KiB = 1024 bytes, 1 MiB = 1024 KiB. These are more accurate for computer memory and file sizes in operating systems.

Bits vs Bytes

1 byte = 8 bits. Network speeds typically use bits (e.g., Mbps = megabits per second), while storage uses bytes (MB, GB). A nibble is half a byte (4 bits).

Storage Units

Blocks and sectors are typically 512 bytes, used in disk storage. Words are 16 bits (2 bytes) in many computer architectures. Modern SSDs often use 4KB sectors.
